es6 精炼

let 和 const

  • let有块作用域
    • 在同一个作用域内，let声明的变量不能重复
    • var 定义的变量会提升
    • let 定义的变量不提升，不能在声明前使用
  • const 有块级作用域
    • 一旦赋值不能改变
    • 声明必赋值
    • let user = {}
      • const PERSON = user
      • user改变，PERSON不会报错，因为PERSON的值是个指针

解构赋值

  • 左右两边结构必须一样

  • 右边必须是个东西

  • 声明和赋值不能分开

  • 解构数组（按照顺序赋值）

  • 解构对象 （按照属性名赋值）

    内置对象解构

  • 解构字符串

字符串方法

  • ‘string’.includes(‘g’，3) //true 从第3个开始匹配
  • ‘string’.startsWidth(‘s’) //true
  • ‘string’.endsWidth(‘g’) //true
  • ‘string’.repeat(3) //‘stringstringstring’

模板字符串

Symbol类型

  • 基本数据类型

    • Number
    • String
    • Boolean
    • undefined
    • null
    • object
    • Symbol

  • 用处：防止对象属性被覆盖

Proxy

  • 定义一个对象时可以对其取值get和赋值set做一些操作

Set

  • 自动去重的数组
  • 一些方法

函数扩展

  • 参数默认值

  • 参数解构赋值

  • rest参数

  • … 扩展运算符

    • 合并数组

  • 箭头函数

    • 不能new

    • 不能通过arguments获取参数列表，但可以通过…args获得

新版面向对象

  • class关键字、构造器和类分开了

  • class里面直接加方法

JSON

  • json对象
  • JSON.stringify(json)
  • JSON.parse(str)
  • 标准写法
    • 只能用双引号
    • 所有名字用双引号引起来

Promise 像同步一样写异步代码

generator函数

  • 普通函数——一路执行到底不能停

  • generator函数——中间能停

  • yield 可以传参

  • yield 可以返回

async await推翻generator

  • 不再依赖runner

  • 支持 =>

  • 在generator里yield的后面异步请求的数据想要给下面代码得依靠runner插件（genObj.next()）
